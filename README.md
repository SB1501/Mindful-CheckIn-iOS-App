# Mindful-CheckIn-iOS-App
An iOS app designed to facilitate regular self-surveys where users can check in on their wellbeing and determine areas to prioritise in their life to improve on self-care.

### Tech Stack
Apple Swift, SwiftUI

### Running Locally (via Simulator)
1. Install Xcode 
2. Download iOS Simulator(s) / SDK
4. Hit run

To run on device, you will need to configure your Xcode environment with an Apple Account and team. Refer to Apple's guidance ([here](https://developer.apple.com/documentation/xcode/running-your-app-in-simulator-or-on-a-device)).
